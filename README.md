# Lululemon Sentiment Tracker

A web scraping project that collects Reddit posts mentioning 
Lululemon and analyzes public sentiment trends over time.
Useful for marketers and consumers tracking brand reputation.

## Project Structure
src/       # Python scraping + analysis code  
data/      # Output CSV files  
docs/      # Citations and references  

## Tools Used
- Python `requests` + `BeautifulSoup` — scraping
- `pandas` — data wrangling
- `TextBlob` — sentiment analysis
